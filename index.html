Jadi Domashno

I developed an online store for homemade dishes as a project for a client during my time at a coding bootcamp. My role in the project was to build the back-end using Laravel. The platform allows users to register as either a chef or a gourmet. As a chef, users can edit their profiles, manage their recipes using a CRUD interface, and indicate their availability for orders. As a gourmet, users can add recipes to their cart and place orders. Once an order is placed, the chef receives an email notification. Additionally, an admin dashboard allows for user account management, with the ability to deactivate or activate users as needed.

In this project, user registration is fully functional; however, the email confirmation process is not available as it is integrated with Mailtrap for development purposes.
